## About Bat Theme

Bat is a modern, minimalist Hugo blog theme focused on providing the best reading and writing experience.

### Design Philosophy

- **Simplicity First**: Remove the unnecessary, keep the essential
- **User Friendly**: Intuitive interaction and configuration
- **Modern Tech**: Built with the latest web technologies
- **Responsive Design**: Perfect display on any device
