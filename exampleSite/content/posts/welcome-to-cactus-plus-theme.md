## Quick Start

### 1. Install the Theme

```bash
cd your-hugo-site
git clone https://github.com/YOUR_USERNAME/hugo-theme-bat.git themes/bat
```

### 2. Configure Your Site

Set the theme in your `config.toml`:

```toml
theme = "bat"
```

### 3. Start the Server

```bash
hugo server -D
```

Visit `http://localhost:1313/` to see the result!
